# n8n---AI-chatbot

## Overview
Created AI chat agent using n8n

## Workflow

Chat message trigger → AI agent → Gemini Chat model → Auto Response

## Tech Stack

* n8n
* Gemini model API
* AI agent
* Chat trigger
* Calculator tool
* Thinking tool

## Screenshots

See the screenshots folder for workflow examples.

## Importing the Workflow

1. Download the JSON file.
2. Open n8n.
3. Click Import Workflow.
4. Upload the JSON file.
